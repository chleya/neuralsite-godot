# 实际量与资源投入起始填报包

用途：
- `work_areas.csv`：当前短编码工作面清单
- `quantities.csv`：由设计量自动展开的实际工程量填报起始表
- `resource_logs.csv`：按工作面展开的人机料填报骨架

建议：
- 在 Excel 中填写 `actual_quantity`、`quantity`、`record_day` 等真实值
- `id` 列留空即可，系统会自动创建
- `work_area_id`、`resource_type` 不要随意改写

- 工作面数量: 12
- 实际工程量起始条目数: 725
- 资源投入骨架条目数: 36
